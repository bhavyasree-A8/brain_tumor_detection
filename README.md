# Brain Tumor Detection using MATLAB

## Overview
This project uses Transfer Learning with ResNet50 to classify MRI brain scans into Tumor and Non-Tumor categories.

## Dataset
Brain MRI Images for Brain Tumor Detection (Kaggle)

- Tumor Images: 155
- Non-Tumor Images: 98
- Total Images: 253

## Technologies
- MATLAB
- Deep Learning Toolbox
- ResNet50
- Image Processing Toolbox

## Results
- Accuracy: 89.33%
- Training/Test Split: 70:30
- Optimizer: Adam

## Features
- MRI Classification
- Tumor Detection
- Confusion Matrix Analysis
- Tumor Region Highlighting
